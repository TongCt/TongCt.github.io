这道题有一个思路就是构建一个滑动的窗口，例如[i,j]是一个窗口，在字符串上滑动，先让j++如果s[j]不在在我们建立的set里，就把它加加进去，同时j往下滑，如果j在这个窗口里，说明有重复的字符串，我们让i往下滑，记录最长的字符串大小。
